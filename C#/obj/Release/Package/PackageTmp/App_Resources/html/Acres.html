REGLA 47.5: REQUISITOS PARA MITIGACIONES Y SIEMBRA DE ÁRBOLES
Sección 47.5.1: Requisitos mínimos de siembra
En todo proyecto y/o actividad de construcción se deberá sembrar un mínimo de diez (10)
árboles por cada cuerda que sea impactada por el proyecto o actividad de construcción
